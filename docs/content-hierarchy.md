# 内容父子层级设计

## 背景

词汇表、句子题组和台词材料不再只按平铺列表使用。HSK1、HSK2、水果词汇等可以作为一级学习来源；HSK1 第1课、第2课等作为子级。父级本身仍然是可学习对象。

## 学习范围规则

- 选择父级时，学习范围包含父级自身和所有启用子级。
- 选择子级时，只学习该子级范围。
- 词汇学习、句子练习、连连看、消消乐和台词训练统一使用该范围规则。
- 停用的父级不会出现在学生端；父级停用时，后台对子级内容只允许查看，禁止对子级词汇条目、句子题、台词行或台词词汇解析做新增、编辑、停用、绑定媒体等操作。

## 全局内容库与归属

- 词汇条目是全局词库内容，同一个词只维护一条记录。
- 一个词汇条目可以归属到多个词汇表，例如“中国”可同时归属“HSK1 第1课”和“国家词汇”。
- 句子题是全局题库内容，同一个句子题只维护一条记录。
- 一个句子题可以归属到多个题组，例如同一句子可同时用于“HSK1 第1课”和“天气表达”。
- 后台新增或 CSV 导入句子题时不强制先选择题组；没有显式归属时，系统自动归属到“听音频排序、按拼音排序、听写汉字、拼音写句”四个默认题组。
- 后端通过 `vocab_list_items` 和 `exercise_set_items` 维护归属关系；旧的 `vocab_list_id`、`exercise_set_id` 仅作为兼容主归属字段保留。
- 后台可以在全部词库/题库中搜索，也可以在某个词汇表/题组及其子级范围内搜索。
- 学生端学习某个父级时，按归属关系读取父级与启用子级内容，同一条内容在同一学习范围内只出现一次。

## 前端交互

- 有子级的词汇表进入后显示两个选项卡：全部学习、按课学习。
- 句子练习先选择句子题组；题组本身决定练习方式，进入父级题组后只显示“全部练习、分组练习”两个范围选项，不再二次选择玩法。
- 有子级的台词材料进入后显示两个选项卡：全部训练、按课训练。
- 游戏选择词汇来源时，父级展示为“名称 (全部)”，子级展示为“父级 / 子级”。
