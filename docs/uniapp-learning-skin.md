# UniApp 学习皮肤

## 选择

移动端采用 `Native Learning + Farm Control` 学习 App 皮肤：页面结构保持主流移动 App 的干净层级，按钮和关卡控件参考沪江词场/词汇农场一类学习 App 的轻游戏化质感，让操作控件有按压感和成长感。

## 设计原则

- 背景使用干净浅灰蓝，不做强装饰网格，保证手机端像原生 App。
- 主行动使用稳定玉绿色，辅助状态使用蓝色、暖黄色和红色分层。
- 卡片保持白底、大圆角、轻阴影，减少彩条和纸面纹理。
- 学习页重点突出内容本身，控制按钮保持清楚但不抢视觉。
- 主按钮使用叶绿色渐变、底部厚度和轻高光，避免纯色块。
- 次按钮使用奶白渐变、底部浅边和轻阴影，避免普通胶囊。
- 词块、分组、课程入口等学习控件统一做成可按压小牌，保留轻游戏化反馈。
- 进度条、会员标识和状态标签不使用按钮样式，避免看起来可以点击。
- 登录、首页、词汇、练习、收藏等页面使用统一按钮高度、圆角、字重和按压反馈。
- 汉字学习内容继续使用内置楷书字体，保证手机端展示稳定。

## 实现位置

- 全局皮肤变量和覆盖层：`mobile/xc-uniapp/src/App.vue`
- 语言切换按钮适配：`mobile/xc-uniapp/src/components/LanguageSwitch.vue`
- 导航栏和 TabBar 基础颜色：`mobile/xc-uniapp/src/pages.json`
- 首页、功能页、词汇列表、背单词、句子练习、收藏页、登录页均有页面级样式补强。
