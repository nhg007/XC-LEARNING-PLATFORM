# 内容 CSV 导入设计

## 目标

CSV 模板面向运营和内容编辑人员，不直接暴露数据库字段名。用户只填写业务上能理解的列；父子关系优先由后台页面上下文带入，避免在 CSV 中要求用户填写 `vocab_list_id`、`exercise_set_id` 等数据库外键。

## 模板列头

- 新下载的模板第一行使用中文业务列名，例如“汉字/词语”“英文释义”“状态”。
- 后端导入仍兼容旧英文列头，例如 `hanzi`、`meaning_en`、`status`，避免历史模板立即失效。
- 模板中的“记录ID（更新时填）”留空表示新增；填写已有记录 ID 表示更新。
- 状态可填“启用/停用”或 `active/inactive`。
- 词表类型、题组题型、材料类型支持中文业务枚举，也兼容后端枚举值；句子题不再要求填写题型。
- 音频素材ID、中文音频素材ID、封面素材ID、词汇笔画演示素材ID均可留空，也可整列不提供；导入后可在列表中上传媒体并手动或批量绑定。

## 父级上下文

以下子内容可以由后台页面传入所属对象：

| 导入类型 | 所属对象 | CSV 中是否填写外键 |
| --- | --- | --- |
| 词汇条目 | 词表 | 可不填写；有 `contextId` 时自动归属，没有时作为全局词汇导入 |
| 句子题 | 题组 | 可不填写；有 `contextId` 时自动归属，没有时默认归属四个系统题组 |
| 台词行 | 台词材料 | 不填写，由 `contextId` 指定材料 |
| 词汇解析 | 台词行 | 不填写，由 `contextId` 指定台词行 |

如果通过接口直接导入且没有传 `contextId`，词汇条目和句子题可以不带归属；也可使用旧外键列兼容导入并自动建立归属。台词行和词汇解析仍需要明确所属对象。

## 后台交互

- 内容管理使用选项卡展示词汇表、词汇条目、媒体资源、题组、句子题、台词材料、台词行和词汇解析。
- 父级下拉保留停用数据；如果父级已停用，子表数据仍可查看，但新增、编辑、导入、批量绑定、启停和删除等子级操作不可用。
- 下载词汇条目和句子题模板时，可以不选择所属对象；选择后模板会省略所属列并由系统带入归属。
- 上传词汇条目和句子题 CSV 时，可以不选择所属对象；词汇条目未选择时导入为全局内容，句子题未选择时默认归属“听音频排序、按拼音排序、听写汉字、拼音写句”四个系统题组，后续可在编辑弹窗中归属到一个或多个词汇表/题组。
- 台词行和词汇解析下载或上传前，仍必须选择所属对象；父级停用时不允许下载或导入。
- 后端以弹窗选择的 `contextId` 为准；如果 CSV 里额外包含旧外键列且与本次选择不一致，导入该行失败。
- 导入结果按行返回成功数和错误列表，错误信息使用业务列名而不是数据库字段名。
- 错误行号使用 CSV 文件中的真实行号，模板里的 `#` 说明行不会影响行号判断。
- 前台对导入失败使用弹窗展示完整错误列表；如果部分行已经成功，提示用户重新导入时只保留失败行，避免重复新增。
